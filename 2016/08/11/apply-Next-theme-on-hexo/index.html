在hexo上应用Next主题

发表于 2016-08-11   |   分类于 教程   |     |  

嫌弃hexo自带的主题？没事，刚好，我也嫌弃。那就换个主题吧。

我用的Next主题，很简洁、淡雅的一款主题。主题安装这里介绍的很清楚。我就不再絮叨。移步Next主题官方文档。
我对主题进行了一些简单的自己觉得是优化的改动。

字体大小

文章字体大小

原主题文章字体是14px，太小了。我给改成18px
文件地址：F:\myBlog\themes\next\source\css\_variables\base.styl

代码块字体大小

将代码块字体由原有的14px–>16px

代码块字体改成16px之后代码块会出现滚动条很难看，还要改代码块字体行高以此来保持整个代码块的美观。
